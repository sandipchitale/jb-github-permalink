# GitHub Permalink

This plugin supports the following two actions in the editor context Open In sub menu as well as the line gutter context menu in any of the JetBrains IDEs (IntelliJ, WebStorm, PyCharm ...):

- Copy GitHub Permalink - copies the GitHub permalink for the current line in the editor. Good for pasting in email or Slack

```
https://github.com/sandipchitale/jb-github-permalink/README.md
```

- Open on GitHub - Opens file at the current line on GitHub using the Permalink
